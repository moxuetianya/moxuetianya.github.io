# 5. 802.11ac：（2013 年）使用 5GHz 频段，最高速度为 6.9Gb/s，采用 MU-MIMO 技术提高多设备连接效率。
# 6. 802.11ad：使用 60GHz 频段，最高速度为 7Gb/s，适用于高带宽、低延迟的应用程序。
# 7. 802.11af：使用 54-790MHz 频段，适用于低功耗、长距离的物联网应用。
# 8. 802.11ah：使用 0.9-1.17GHz 频段，适用于低功耗、长距离的物联网应用。
# 9. 802.11ax（Wi-Fi 6）：使用 2.4GHz 和 5GHz 频段，最高速度为 9.6Gb/s，采用 OFDMA 技术提高多设备连接的效率和信号质量。
