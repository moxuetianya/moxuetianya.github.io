qemu-user 启动失败排查

十一月 25, 2025 #FAQ #Frequently Asked Questions

fuzz模糊测试使用afl qemu trace 失败

怀疑是环境变量导致额的， 最后定位到环境变量

...
"-E" "TMUX=/tmp/tmux-1000/default,18190,1" 
...

实验发现，是环境变量中的值存在‘，’ 就会启动失败

去掉','， 是可以正常启动的

...
"-E" "TMUX=/tmp/tmux-1000/default" 
...

# 或者
unset TMUX
unset no_proxy
