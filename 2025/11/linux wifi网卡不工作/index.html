linux wifi网卡不工作

十一月 23, 2025 #wifi

现象

ip link
iw dev
# 这些命令 都看不到对应的无线网卡
# pci设备是可以看到wifi设别的
# lspci -v | grep -i wifi

排查过程

sudo apt update
sudo apt install firmware-iwlwifi

lspci -v | grep -i wifi
sudo lspci -v -s 00:14.3

rfkill list all
nmcli radio wifi
nmcli radio wifi on

sudo lsmod | grep iwl
sudo dmesg | grep -i iwl

# 查看系统是否至少识别 Wi-Fi 为网络设备
sudo lshw -C network
# 如果无线网卡显示：
# UNCLAIMED
# 驱动没有绑定成功
ls /sys/class/net/

重新加载内核模块后恢复了？

# 1. 清空当前的 dmesg 缓冲区，排除干扰
sudo dmesg -C
# 2. 卸载驱动模块
sudo modprobe -r iwlwifi
# 3. 重新加载驱动模块 (这一步会触发初始化)
sudo modprobe iwlwifi
# 4. 立即查看刚刚产生的日志 (这次肯定会有内容！)
sudo dmesg
