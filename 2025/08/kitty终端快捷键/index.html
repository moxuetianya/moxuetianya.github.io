kitty终端快捷键

八月 14, 2025 #kitty #终端

kitty 终端模拟器有许多快捷键，主要用于导航、窗口管理、文本操作和配置等。以下是一些常用的快捷键： 基本导航:

  • Ctrl+Shift+Enter: 新建标签页﻿
  • Ctrl+Shift+W: 关闭标签页﻿
  • Ctrl+Shift+PageUp / Ctrl+Shift+PageDown: 切换标签页﻿
  • Ctrl+Shift+左箭头 / Ctrl+Shift+右箭头: 切换窗口﻿
  • Ctrl+Shift+方向键: 在当前窗口中移动光标﻿
  • Ctrl+Shift+T: 在当前窗口中打开一个新的标签页﻿
  • Ctrl+Shift+Q: 关闭当前窗口﻿
  • Ctrl+Shift+R: 重新加载配置文件﻿

文本操作:

  • Ctrl+Shift+C: 复制选中文本
  • Ctrl+Shift+V: 粘贴文本
  • Ctrl+Shift+X: 剪切选中文本
  • Ctrl+Shift+Z: 撤销
  • Ctrl+Shift+Y: 重做
  • Ctrl+Shift+方向键: 在文本中移动光标
  • Ctrl+Shift+PageUp / Ctrl+Shift+PageDown: 在文本缓冲区中向上/向下滚动﻿

其他快捷键:

  • Ctrl+Shift+F2: 编辑kitty 配置文件
  • Ctrl+Shift+F5: 重新加载kitty 配置文件
  • Ctrl+Shift+F6: 调试kitty 配置文件
  • Ctrl+Shift+Escape: 打开kitty 终端
  • Ctrl+Shift+加号 / Ctrl+Shift+减号: 调整字体大小
  • Ctrl+Shift+F: 在当前窗口中查找文本﻿

自定义快捷键:

  • 可以在 kitty.conf 文件中自定义快捷键。例如，要将新建标签页的快捷键设置为 Ctrl+Alt+T，可以在 kitty.conf 中添加以下行：
map ctrl+alt+t launch --cwd=current

查找更多快捷键:

  • 可以在 kitty 文档 中查找更多快捷键和配置选项

配置kitty字体： 具体应用场景

sudo apt install fonts-jetbrains-mono
cat <<EOF >> ~/.config/kitty/kitty.conf
font_family JetBrains Mono
font_size 14.0
EOF

配置alacritty字体大小

cat <<EOF >> ~/.config/alacritty/alacritty.toml
[font]
size = 18
EOF
