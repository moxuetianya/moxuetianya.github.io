远程桌面配置

独立会话 vs 屏幕共享

模式特点
独立会话一台机器多人同时使用，各自会话隔离（RDP 支持）
屏幕共享多人共享同一会话，抢占键鼠焦点（moonlight/parsec）

Wayland 方案

Waypipe + SSH（应用级转发）

适用于远程运行单个 Wayland 原生应用，低带宽环境友好。

# 安装
sudo apt install waypipe openssh

# 启动远程应用
waypipe -c lz4=9 ssh user@remote firefox

# 多线程渲染 + 无光标传输（减少带宽）
waypipe -j 4 -c lz4=9 ssh user@remote blender
waypipe -C none ssh user@remote gedit

限制：不转发 XWayland 应用；客户端和服务端 Wayland 协议版本需一致。

GNOME Remote Desktop (g-r-d)

  • GNOME 内置 RDP 服务端
  • GNOME 47 起支持持久化会话（重连时不注销）
  • GNOME 46 及以下重连会注销当前会话

配置文件：~/.config/gr-d/

Weston RDP

企业级 RDP，支持 Windows mstsc.exe 直接连接。

~/.config/weston.ini：

[screen-share]
module=rdp.so
address=0.0.0.0
port=3389

[core]
gbm-format=argb8888

[rdp]
color-depth=32
weston --backend=rdp-backend.so

Krdp

KDE Plasma 原生 RDP，仅限 KDE 环境。

wprs

Rootless Wayland RDP 服务端。

X11 方案

方案特点
x11 转发（SSH -X）低带宽，单个应用
xrdp无脑首选，兼容性最好
FreeRDP高性能 + RDP 扩展
x2go多会话支持
ThinLinc商用方案

xrdp 认证弹窗修复

创建 /etc/polkit-1/localauthority.conf.d/02-allow-colord.conf：

polkit.addRule(function(action, subject) {
  if ((action.id == "org.freedesktop.color-manager.create-device" ||
       action.id == "org.freedesktop.color-manager.create-profile" ||
       action.id == "org.freedesktop.color-manager.delete-device" ||
       action.id == "org.freedesktop.color-manager.delete-profile" ||
       action.id == "org.freedesktop.color-manager.modify-device" ||
       action.id == "org.freedesktop.color-manager.modify-profile") &&
       subject.isInGroup("{users}")) {
    return polkit.Result.YES;
  }
});

GNOME 47 RDP 已知问题

  1. gnome-shell 加载 NVIDIA 驱动后调用 MESA，增加闪退概率
  2. 幽灵账户 + u→g 组权限：firefox/edge 无法保持 cookie/localstorage（可能与 keyring 相关）
  3. aRDP 客户端尚未支持 Super Key
