clash verge 与 233boy sing-box 配合使用

十月 06, 2025 #clash verge #sing-box

sing-box 服务端配置

在vps 上配置一个代理节点

bash <(wget -qO- -o- https://github.com/233boy/sing-box/raw/main/install.sh)
sing-box

安装clash verge 客户端


wget https://github.com/clash-verge-rev/clash-verge-rev/releases/download/v2.4.2/Clash.Verge_2.4.2_amd64.deb
sudo apt install ./Clash.Verge_2.4.2_amd64.deb

配置clash verge 添加自建节点

  1. 在clash的订阅页面新建一个local本地配置,文件可以置空

  2. 右击编译本地配置文件 配置模板

  3. 右击编辑本地节点 将服务端的配置复制进来即可 复制【链接（RUL）】

# 查看服务端配置
sb info
